JavaScript Compiler Build and Deploy Documentation

I. Overview

This document covers the basic steps needed to build and deploy the JavaScript compiler software.

II. Clone the source

Visit https://github.io/chriskwlee/Compiler for the most up-to-date version of the JavaScript compiler.

Alternatively, get the most up-to-date source code from Christopher Lee's GitHub code repository.

git clone https://github.com/chriskwlee/Compiler

III. Running

Open the corresponding HTML file linked to the JavaScript compiler. Insert code to be compiled, then run.
