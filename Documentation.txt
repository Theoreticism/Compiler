CCompiler Build and Deploy Documentation

I. Overview

This document covers the basic steps needed to build and deploy CCompiler, a web-based compiler written in Javascript.

II. Clone the source

Visit https://github.io/chriskwlee/Compiler for the most up-to-date web version of CCompiler.

Alternatively, get the most up-to-date source code from Christopher Lee's GitHub code repository at https://github.com/chriskwlee/Compiler.

III. Running

Open the corresponding HTML file linked to the JavaScript compiler. Insert code to be compiled, then run.
